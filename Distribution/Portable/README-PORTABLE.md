# BlackNotepad - Portable Version

## About
BlackNotepad is a dark theme notepad for Windows. This is the portable version - no installation required.

## How to Use
1. Extract all files to a folder of your choice
2. Run `BlackNotepad.exe`
3. That's it!

## Features
- Dark theme UI
- Multiple file formats supported (.txt, .md, .json, .xml, .csv, .html, .js, .py, etc.)
- Multiple font colors and zoom levels
- File association support (double-click .txt files to open)
- Window position/size persistence
- Print functionality

## Notes
- Settings are stored in: `%APPDATA%\BlackNotepad`
- For full installation with Start Menu shortcut, use the installer version
